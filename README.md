# spin-sdk

SDK for developing Fully On-Chain Games by zkSpin.

## Installation

```shell
npm install --global @zkspin/sdk
```

## Docs

https://zkspin.gitbook.io/zkspin

### Trouble Shooting

#### For Windows:

> use Windows Subsystem for Linux (WSL)
